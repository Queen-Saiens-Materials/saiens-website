# Saiens 網站內容編輯指南（給編輯同事）

> 你不需要懂程式。照下面步驟操作即可，任何改動都有版本紀錄可還原，不用怕改壞。

## 登入後台

1. 打開 `https://www.saiens.group/admin`
2. 用你收到邀請的帳號登入（Tina Cloud）
3. 左側選單就是所有可編輯的內容

## 修改文章（News）

1. 左側選「文章（台灣）」或「記事（日本）」
2. 點要修改的文章 → 右側是編輯器、可即時預覽
3. 內文編輯器支援：標題、粗體、連結、插入圖片（直接拖曳上傳）
4. 改完按右上角 **Save** → 約 1-2 分鐘後自動上線

### 新增文章

1. 進入文章列表 → 右上 **Create New**
2. 必填：標題、日期（格式 2026-07-12）、內文；建議附：摘要、縮圖
3. Save 後同樣自動上線；舊網址轉址與網站地圖都會自動更新，不用做任何事

## 修改首頁文案 / 圖片

1. 左側選「首頁（台灣）」「首頁（日本）」或「首頁（美國）」
2. 依區塊展開欄位（Hero、One Stop Service、產品、品牌、認證……），直接改文字
3. 換圖：點圖片欄位 → 上傳新圖（建議寬度 ≥1600px、JPG，單檔 <2MB）
4. Save → 自動上線

## 修改聯絡資訊（地址 / 電話 / 營業時間 / 社群連結）

左側選「網站設定」→ 展示間資訊或社群連結 → 改完 Save。
全站引用處（頁首、頁尾、Visit Us 頁）會一起更新。

## 注意事項

- **Save = 發布**：按下儲存就會上線，發布前請確認內容
- **不要改的東西**：文章的網址代稱（slug）發布後不要再改，會讓已分享出去的連結失效
- 圖片請用照片原檔上傳，不要截圖（畫質差）
- 改壞了不用慌：所有版本都有紀錄，聯絡 Michael 可以還原任何一次修改

## 常見問題

**Q：按了 Save 網站沒變？**
等 2 分鐘再重新整理（部署需要時間）；還是沒變就清瀏覽器快取或換無痕視窗看。

**Q：可以排程發布嗎？**
目前不行，Save 即發布。想先寫好草稿：寫在別處（如 Google Docs），確定要發時再貼上。

**Q：編輯器打不開 / 登入有問題？**
聯絡 Michael（michael@saiens.us）。
